# Wireframe: Journey 5 — LINE Order (Remote Customer)

## ผู้ใช้: พนักงานหน้าร้าน (Counter Staff) + ลูกค้าผ่าน LINE
## Scenario: ลูกค้าสั่งงานผ่าน LINE ไม่ได้มาหน้าร้าน ส่งงานทางขนส่ง
